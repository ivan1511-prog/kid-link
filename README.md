# KidLink

Система оповещения о расстоянии на часах LilyGO T-Watch-S3. Обе пары вибрируют, когда удаляются друг от друга на ~2 метра.

## Зачем

Контроль ребёнка на прогулке, в торговом центре, на пляже — без телефона и интернета.

## Как работает

```
👨 Родитель          👦 Ребёнок
   T-Watch    ← BLE →   T-Watch
      ↓                    ↓
   вибро ✓              вибро ✓
```

Каждые часы — одновременно маяк и сканер. Измеряют RSSI сигнала друг друга и вибрируют, если расстояние превышает порог.

## Статусы

| Состояние | Что происходит |
|-----------|----------------|
| 🟢 OK | Всё хорошо, ребёнок рядом |
| 🔴 FAR | Далеко (~2-5 м) — вибрация каждые 3 сек |
| ⚠️ NO SIGNAL | Нет связи — вибрация каждые 1 сек |

## Железо

- 2× [LilyGO T-Watch-S3](https://www.lilygo.cc/products/t-watch-s3)
- Встроенный BLE (ESP32-S3)
- Встроенный вибромотор

## Стек

- PlatformIO + Arduino
- [TTGO_TWatch_Library](https://github.com/Xinyuan-LilyGO/TTGO_TWatch_Library)
- [NimBLE-Arduino](https://github.com/h2zero/NimBLE-Arduino)
